# Privacy Policy

*Last updated: 2026-05-15*

This service is a personal automation tool operated by Rob Cannon for the sole
benefit of the operator. There are no third-party subscribers.

## Data collected

- Mobile phone number of the operator (for SMS routing).
- Inbound message content sent by the operator.
- Outbound message content generated by the system.

## Data use

Data is used solely to route messages between the operator and the operator's
personal automation system. No data is sold, shared, or transferred to third
parties for marketing or any other purpose.

## Data retention

Message content is retained in local logs for operational debugging and is
deleted at the operator's discretion. There is no long-term cloud storage of
message content.
